# Old-Driver-Protocol-AndroidDemo
这是一个Android端的ODP加密解密Demo 
  
提供了基于Fast-ODP协议的加密解密  


# How to use  
开启ODP服务  
  
复制ODP消息体并在弹窗内输入密钥  
  
获得内容  

*提供EN/DE测试按钮来直接进入`TestActivity`*
